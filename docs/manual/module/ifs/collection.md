# 模块 collection
基础容器模块

引用方法：
```JavaScript
var collection = require("collection");
```
## 对象
        
### Map
创建一个 Map 容器，参见 Map
```JavaScript
Map collection.Map;
```

### List
创建一个 List 容器，参见 List
```JavaScript
List collection.List;
```

### Queue
创建一个 Queue 容器，参见 Queue
```JavaScript
Queue collection.Queue;
```

Module Management

Fibjs’s module management uses the same machanism with CommonJS. The modules could be loaded synchronously and asynchronously, which will make it compatible with both frontend and backend codes.

Basic Elements

There are four basic ways to load a module, including two functions (require() Function and define() Function) and two objects (module Object and ref obj_exports).

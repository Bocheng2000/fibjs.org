static Stream websocket.connect ( String  url,

)
static

创建一个 websocket 连接，并返回握手成功的 Stream 对象

参数
url指定连接的 url，支持 ws:// 和 wss:// 协议
origin指定连接的授权域名
返回
返回连接成功的 Stream 对象，可能为 Socket 或者 SslSocket
